MarkDown

最近用github+hexo搭建了一个自己的博客，也算是有了自己的家了，以前经常想写一些东西记录一些笔记不知道去哪里写。

1. 什么是markDown

Markdown 是一种轻量级标记语言，创始人为约翰·格鲁伯（John Gruber）和亚伦·斯沃茨（Aaron Swartz）。它允许人们“使用易读易写的纯文本格式编写文档，然后转换成有效的XHTML(或者HTML)文档”

说那么多，其实就是写简单的文本能够转换成html或者pdf文件。
.md文件中可以嵌套html代码

我是用的工具是 MarkdownPad 2 可以实时预览便宜的html样式 适合新手！

2. github+hexo搭建博客基本原理

大致过程就是利用node封装的hexo将本地写好的.md文件编译成html 然后hexo结合主题文件生成html css js静态文件 最后上传github Pages 静态资源服务器
