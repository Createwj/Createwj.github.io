虽然计算属性在大多数情况下更合适，但有时也需要一个自定义的 watcher。这是为什么 Vue 通过 watch 选项提供一个更通用的方法，来响应数据的变化当你想要在数据变化响应时，执行异步操作或开销较大的操作，这是很有用的。
